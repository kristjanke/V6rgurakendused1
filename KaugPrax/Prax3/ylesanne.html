Lühidalt CSS-ist

  • CSS-is viidatakse HTML elementidele ning antakse neile stiilikirjeldusi. Elementidele viitamiseks kasutatakse selektoreid.
    Erinevate selektorite kohta loe siit
  • CSS-iga saab anda HTML elementidele igasuguseid omadusi.
    Nimekiri erinevatest CSS-iga muudetavatest omadustest siin
  • tüüpiline CSS näeb välja taoline:
    selektor {
      omadus1:väärtus;
      omadus2:väärtus;
    }
    selektor2, selektor3 {
      omadus:väärtus;
    }

Ülesanded

  1. kui soovid eelmise praktikumi seisu jäädvustada, kopeeri eelmise praktikumi tulemus uude kausta ning jätka oma tegevust seal
  2. Ava pealeht.html ning:
    • tekita esimesel korral lisatud pildile väiksem versioon. Selleks kasuta GIMP-i või mingit muud pilditöötlustarkvara.(Vali mingi sobiv formaat nt 150x100 px. Ürita pilti mitte väga välja venitada, väiksem versioon võib ka olla vaid osa suurest pildist). Kui ei oska, siis loe siit ja siit
      kui pilditöötlustarkvara puudub
    • kuna pilte saab olema palju, siis on selguse mõttes parem nad eraldi kaustadesse paigutada. Selleks tekita kaks uut kausta: small ja big ning liiguta pildid neisse.
    • asenda pealehel suur pilt väiksega (NB! aadress on nüüd natuke teine, kui ei mäleta vaata esimese loengu slaide) ning muuda see väike pilt lingiks suurele pildile (kui oled segaduses siis vaata kuidas validaatori logo tehtud on)
    • lisa veel paar pilti
    • Kontrolli, et (http://validator.w3.org) on ikka veel rõõmus ja roheline.
  3. lisa head tagide vahele stiili kirjeldamiseks vajalik style element (selle vahele saab kirjutada CSS-i)
  4. kujunda pealkirjale sobiv värv (color).
    • Selleks, et kõik tulevased selle taseme pealkirjad samat värvi ei oleks kasuta css-is selektorit #pealkiri mis viitab konkreetsele id atribuudi väärtusele.
    • Värvi saab kirjutada kas #HEX NUMBRID või rgb(decimal numbrid). Näiteks #F8F8FF ja rgb(248,248,255) on ekvivalentsed Värvikaardi leiad siit
  5. muuta menüülinkide stiili (kasutada nt: color, text-decoration, font-weight, font-size jne.). Kasutada selektorit #menu a piiramaks stiili mõju menüü sees olevatele linkidele
  6. määrata stiil ka linkidele, mille kohal parasjagu hiirega ollakse (et neid rohkem esile tuua). Selleks kasutada selektorit #menu a:hover, kus :hover ütleb, et hiirega ollakse elemendi kohal
  7. määra kogu lehe taustavärv (background) ning joonda kõik elemendid keskele (text-align).
    Selleks, et kõigi elementide sisu oleks keskele joondatud piisab neid ümbritseva elemendi (body) stiili kirjeldamisest
  8. kirjelda div-ile #content järgmine kujundus:
    • taustavärv valge (#ffffff)
    • padding 20px
  9. Organiseerida nii, et piltidel on raam (border) ümber, aga validaatori nupul seda ei ole.
    Vihje: kirjelda stiil ainult kindla div-i sees olevatele img elementidele või anna piltidele class atribuut.
  10. Kui kasutad windowsi, saad kontrollida kas IE-s validaatori logol on raam või mitte. Selle saab eemaldada andes validaatori logole omaduse border:none;
  11. Kontrolli taaskord, et (http://validator.w3.org) on ikka veel rõõmus ja roheline, seda saad teha lihtsalt: kliki lehel olevale validaatori logo pildile.
  12. Lisaks saad kontrollida CSS õigsust CSS validaatoris. Edaspidiseks mugavaks CSS valideerimiseks võid (aga ei pea) pealeht.html HTML validaatori logo juurde lisada ka CSS validaatori logo:
